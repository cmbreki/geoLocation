# where are U now-app

A simple project for location detection using the HTML5 Geolocation API.

### Dependencies

##### Front-end  
- AngularJS 1.5.8  
- Angular-animate 1.5.8  
- ngGeolocation  0.0.8  
- Angular-simple-logger 0.1.7  
- Leaflet 0.7.7  
- Leaflet-plugins 1.6.1
- ui-leaflet 1.0.2
- ui-leaflet-layers 0.1.2
- Angular-material 1.1.1
